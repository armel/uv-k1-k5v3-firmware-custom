# Технічне завдання (ТЗ) — DualMode Firmware для UV-K5 V3

## 1. Режими роботи

| Режим | TX | RX |
|-------|-----|-----|
| **EU Safe** | Тільки PMR446 (446.00625–446.19375 MHz) | 18–1300 MHz (весь діапазон) |
| **Blackout** | Будь-яка частота 18–1300 MHz | 18–1300 MHz |

- **За замовчуванням**: завжди EU Safe при увімкненні/перезавантаженні
- **Blackout**: активується через PIN, не зберігається між сесіями

---

## 2. PIN та захист Blackout

- **Довжина PIN**: 4–6 цифр
- **Зберігання**: EEPROM (адреса 0x00A170)
- **3 невдалі спроби** → блокування на 30 с
- **Меню**: пункт «Set PIN» для зміни PIN
- **Екран введення**: MENU — підтвердження, EXIT — скасування

---

## 3. Dual VFO layout

| Рядок | VFO | Призначення | TX | Потужність |
|-------|-----|-------------|-----|------------|
| **Верхній** | VFO A (index 0) | Робочий канал зв'язку | Так (у дозволених межах) | Low1–High |
| **Нижній** | VFO B (index 1) | Сканер/монітор | **Ні** | ---- |

- **PTT завжди передає на VFO A** (верхній рядок)
- **TX на VFO B заборонено** — подвійний біп при спробі
- **Перемикання A/B**: тільки зміна RX/перегляду, TX залишається на VFO A

---

## 4. Індикація

- **Статус-бар**: EU або BL біля індикатора батареї
- **VFO B**: замість рівня потужності — «----»

---

## 5. Меню

**4 категорії**: Radio, Scan, Mode, Config

| Категорія | Приклади пунктів |
|-----------|------------------|
| **Radio** | Step, Power, CTCSS/DCS, FM Radio, ChList, ChSave... |
| **Scan** | ScList, ScPri, ScnRev, NOAA-S |
| **Mode** | OpMode, Set PIN, F1Shrt, F2Shrt... |
| **Config** | KeyLck, TxTOut, BatSav, BLTime, Beep... |

---

## 6. Сканування

- **Повне сканування**: 18–630 MHz → стрибок через мертву зону → 840–1300 MHz → зворот до 18 MHz
- **Custom Range**: діапазон від VFO A до VFO B (коли задано)
- **Scan Lists**: сканування каналів з пам'яті (списки 1–3)
- **Сканування виконується на VFO B**
- **Автомодуляція**: AM для 18–30, 108–137, 225–400 MHz; FM для решти
- **Quick Bookmark**: F+MENU під час сканування — збереження частоти в List 3

---

## 7. Пресети каналів (при першому запуску)

| List | Канали | Зміст |
|------|--------|-------|
| **1** | Ch 1–16 | PMR446 (PMR-01 … PMR-16) |
| **2** | Ch 17–28 | Emergency/Marine (AIR-SOS, MAR-CH16, MIL-SOS, ISS, NOAA...) |
| **3** | Ch 29+ | Порожній, для Quick Bookmark |

---

## 8. Інше

- **FM Radio**: 87.5–108 MHz (ENABLE_FMRADIO)
- **Wide RX**: 18–1300 MHz (ENABLE_WIDE_RX)
- **Прибрано**: AirCopy, Game, Screenshot
- **Юридичне застереження**: при першому запуску після оновлення
- **CMake preset**: DualMode (Bandscope + DUALMODE, без AirCopy/Game/Screenshot)
